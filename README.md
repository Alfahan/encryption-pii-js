## Installation this package to your project

1. Run `npm` or `yarn` to install:
    ```
    npm i pii-agent-js
    ```

2. Set the keys in your `.env` file:
    ```
	CRYPTO_AES_KEY=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
	CRYPTO_HMAC_KEY=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
   ```
